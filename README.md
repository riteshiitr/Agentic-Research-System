# Agentic-Research-System

The Agentic Research System is a multi-agent application designed to automate the end-to-end research process. Unlike traditional chatbots that rely solely on pre-trained knowledge, this system actively gathers real-time information from the internet, analyzes multiple sources, synthesizes findings into structured reports, and performs a self-review to improve output quality.

The project demonstrates how Agentic AI architectures can be used to build systems capable of planning, tool usage, task decomposition, and iterative reasoning.
